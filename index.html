cdn.sankhacooray.com

Shared, public, non-sensitive front-end components for the sankhacooray.com network.

Components

  • /js/sankha-footer.js — shared network footer (auto-theming)
  • /js/sankha-analytics.js — page-view beacon + fame badge
  • /js/sankha-weather.js — token-free weather widget
  • /js/sankha-chat.js — floating chat widget

Usage

Reference any component directly, e.g.
<script async src="https://cdn.sankhacooray.com/js/sankha-footer.js"></script>
